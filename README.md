### Lo Esencial

```bash
# 1. Instalar dependencias
npm install

# 2. Configurar variables de entorno
cp .env.example .env
# Editar .env con secretos (generar con: openssl rand -base64 32)

# 3. Desarrollo
npm run dev

# 4. Validar seguridad
node scripts/security-checklist.js
```
